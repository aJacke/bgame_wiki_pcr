# bgame_wiki_pcr
基于pyppeteer截取公主连结wiki_BWIKI的活动日历

## 安装方法
>pip install -r requirements.txt

之后在你的Hoshino res文件夹下创建bwiki的文件夹即可

***注意事项***

如果安装pyppeteer失败，可以参考[这里](https://blog.csdn.net/qq_41362018/article/details/108010246)的方法

## 使用方法

- 每天5点在启用的群里自动发送

- 或者发送 [**日历**] 即可获取活动日历

- [**更新日历**]可以重新获取当天日历图
